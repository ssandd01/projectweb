Aoba Castle Ruins

ปราสาทอาโอบะ (Aoba Castle Ruins) หรือ ปราสาทเซนได (Sendai Castle) เป็นพื้นที่แสดงส่วนที่ยังคงเหลืออยู่ของปราสาทอาโอบะ ซึ่งเป็นปราสาทที่สร้างขึ้นในปี 1600 โดยดาเตะ มาซามุเนะ (Date Masamune) ไดเมียวผู้โด่งดังในยุคเซนโกคุและผู้ปกครองเมืองเซนไดที่มีอำนาจมากที่สุดในสมัยเอโดะ

ปราสาทอาโอบะนั้นสร้างขึ้นโดยมีจุดประสงค์เพื่อป้องกันเมืองบนภูเขาอาโอบะ ซึ่งมีความสูง 100 เมตรจากเมืองด้านล่าง โชคไม่ดีเท่าไหร่ที่ปราสาทถูกทำลายลงจากผลกระทบของสงครามโลกครั้งที่ 2 ปัจจุบันปราสาทแห่งนี้จึงเหลือเพียงเศษซากกำแพงหินด้านนอกและหอรักษาความปลอดภัย

อย่างไรก็ตาม แม้ว่าตัวปราสาทจะไม่มีอยู่แล้ว แต่จากที่ตั้งปราสาทเดิมเราสามารถมองเห็นทิวทัศน์ที่สวยงามของเมืองเซนไดได้ และยังมีดอกซากุระให้ชมในช่วงฤดูใบไม้ผลิด้วย (สำหรับนักท่องเที่ยวสายความรู้ มีพิพิธภัณฑ์ที่บอกเล่าประวัติความเป็นมาของปราสาทแห่งนี้ด้วยนะ)

Location :

  • 1 คาวาอุจิ, เขตอาโอบะ, เมืองเซนได, จังหวัดมิยางิ, ประเทศญี่ปุ่น 980-0862

How to Travel :

  • รถบัส : ในวันเสาร์ อาทิตย์ และวันหยุดนักขัตฤกษ์เท่านั้น รถบัสมิยาโกะระหว่างชิราอิชิซาโอะและโทกัตตะออนเซ็นจะขยายไปถึงยอดภูเขาคัทตะ (ปล่องโอกามะ)
  • รถยนต์ : ประมาณ 65 นาที จากทางแยกต่างระดับ Murata บนทางด่วน Tohoku Expressway

Contact :

  • 022-354-2023

Hours :

  • สวน : เปิดให้เข้าชมทุกวัน ตลอดเวลา
  • พิพิธภัณฑ์ : เปิดทำการทุกวัน ในเวลา 9:00 - 17:00 น. (ช่วงเดือนพฤศจิกายนถึงเดือนมีนาคม เปิดถึงเวลา 16:00 น.)

Entrance Fee :

  • สวน : ไม่มีค่าเข้าชม
  • พิพิธภัณฑ์ : ค่าเข้าชม 700 เยน

Zuihoden Mausoleum

สุสานซุยโฮเด็น (Zuihoden Mausoleum) เป็นสุสานของไดเมียว ‘ดาเตะ มาซามุเนะ’ (Date Masamune) อาคารสุสานได้รับการออกแบบในสไตล์หรูหราในช่วงยุคโมโมยามะ ซึ่งโครงสร้างหลักเป็นงานไม้สีดำที่ตกแต่งด้วยการแกะสลักและทาสีสันสดใส

ส่วนทางเดินภายในพื้นที่ของสุสานนั้นล้อมรอบไปด้วยป่าไม้สนซีดาร์ ซึ่งเป็นสัญลักษณ์ของประวัติศาสตร์อันยาวนานของตระกูลดาเตะ (อาคารที่เห็นในปัจจุบันเป็นการบูรณะสร้างใหม่ เพราะอาคารเดิมถูกทำลายลงในช่วงสงครามโลกครั้งที่ 2)

สุสานถูกสร้างขึ้นตามความประสงค์ของมาซามุเนะ ดาเตะ ผู้ก่อตั้งตระกูลเซนได อาคารปัจจุบันถูกสร้างขึ้นใหม่ในปี 1979 ตามแบบจำลองของซุยโฮเด็น ก่อนที่มันจะถูกทำลายด้วยไฟในสงคราม หัวสิงโตที่แกะสลักบนเสาและกระเบื้องหัวมังกรบนหลังคาได้รับการบูรณะใหม่ สื่อถึงสไตล์ที่หรูหราและงดงามของวัฒนธรรมโมโมยามะ ห้องโถงคันเซ็น-เด็นที่สร้างโดยดาเตะทาดามุเนะลอร์ดคนที่สอง และห้องโถงเซ็นโนะ-เด็นซึ่งสร้างโดยผู้ปกครองคนที่สาม ดาเตะ สึนะมุเนะก็ตั้งอยู่บนพื้นที่เดียวกัน

Location :

  • 23 โอทะมะยาชิตะ, เขตอาโอบะ, เมืองเซนได, จังหวัดมิยางิ, ประเทศญี่ปุ่น 980-0814

How to Travel :

  • รถไฟและรถประจำทาง : จากสถานี Sendai บนสาย Tohoku Shinkansen และ JR ขึ้นรถบัส Loople Sendai ไปลงที่ป้าย Zuihoden-mae และเดินต่อประมาณ 10 นาทีไปยังจุดหมายของคุณ
  • รถยนต์ : ประมาณ 15 นาทีจากทางแยกต่างระดับ Sendai-Miyagi บนทางด่วน Tohoku Expressway

Contact :

  • 022-262-6250

Hours :

  • เปิดให้เข้าชมทุกวัน 9:00 - 16:50 น.

Entrance Fee :

  • ผู้ใหญ่ : 550 เยน
  • นักเรียนมัธยมปลาย : 400 เยน
  • นักเรียนมัธยมต้นและนักเรียนประถม : 200 เยน

Entsuin Temple

วัดเอ็นสึอิน (Entsuin Temple) เป็นวัดที่สร้างขึ้นข้างๆกันกับวัดซุยกันจิในปี 1647 โดยมีจุดประสงค์เพื่อให้เป็นสุสานของ ดาเตะ มิทสึมุเนะ หลานของไดเมียวดาเตะ มาซามุเนะที่เสียชีวิตตั้งแต่อายุยังน้อย คือ 19 ปีเท่านั้น

และเนื่องจากเป็นวัดที่สร้างขึ้นเพื่อเป็นสุสานของผู้ที่เสียชีวิตตั้งแต่อายุยังน้อย วัดเอ็นสึอินจึงเป็นที่ประดิษฐานของพระโพธิสัตว์เจ้าแม่กวนอิม ผู้เป็นสัญลักษณ์แห่งเมตตา เพื่อให้วิญญาณของผู้วายชนม์ได้รับความเมตตาในโลกหน้า

จุดเด่นของวัดแห่งนี้คือสวนหิน เนื่องด้วยการจัดหินและมอสภายในสวนแห่งนี้มีเอกลักษณ์อยู่ที่การใช้อ่าวมัตสึชิมะเป็นต้นแบบ

นอกจากที่นี่จะวิวสวยแล้ว สำหรับสายมูที่ชอบกำไลข้อมือ นอกจากจะมีกำไลขายตามที่เห็นในภาพแล้ว ที่นี่ก็ยังสอนร้อยกำไลให้กับผู้ที่สนใจอีกด้วย โดยจะมีการอธิบายตั้งแต่วิธีร้อยหินและประเภทของหิน เช่น หินแต่ละแบบมีพลังอะไรบ้าง เป็นต้น

ในฤดูใบไม้ผลิ วัดเอ็นทสึอินจะมีการประดับไฟไลท์อัพในเวลากลางคืน ความงามของใบไม้เปลี่ยนสีเล่นแสงไฟยามค่ำคืนนั้นเกินจะบรรยาย อีกทั้งยังมีมุมให้ได้ลองประดิษฐ์ลูกประคำอีกด้วย (มีค่าใช้จ่ายเพิ่มเติม)

Location :

  • 67 มัตสึชิมะ, เขตมิยางิ, เมืองมัตสึชิมะ, จังหวัดมิยางิ, ประเทศญี่ปุ่น 981-0213

How to Travel :

  • รถไฟและรถประจำทาง : จากสถานี JR Sendai ขึ้นรถไฟไปลงที่สถานี Matsushima Kaigan แล้วเดินต่อจากสถานีไปที่วัดโดยใช้เวลาอีกประมาณ 10 นาที

Contact :

  • 022-354-3206

Hours :

  • วันธรรมดา : 9:00 - 15:30 น.
  • วันเสาร์ / วันอาทิตย์ / วันหยุดนักขัตฤกษ์ : 9:00 - 16:30 น.

Entrance Fee :

  • ตอนกลางวัน : 300 เยน
  • เปิดไฟตอนกลางคืนมีค่าเข้าชม : 500 เยน

Fukuurajima Island

เกาะฟุคุอุระจิมะ (Fukuurajima Island) เป็นเกาะเล็กๆใน ‘จังหวัดมิยากิ’ ที่สามารถใช้เวลาเดินชมได้ภายใน 2 ชั่วโมง โดยพื้นที่ส่วนใหญ่ของเกาะจะอยู่ในรูปแบบของสวนสาธารณะ ซึ่งเป็นสวนพฤกษศาสตร์ธรรมชาติที่มีต้นไม้ ดอกไม้ และพืชต่างๆอยู่กว่า 300 ชนิด อีกทั้งยังมีจุดชมทัศนียภาพอันงดงามตลอดแนวทางเดิน จุดเด่นของเกาะแห่งนี้คือสะพานฟุคุอุระบาชิ สะพานสีแดงสดยาว 252 เมตรที่พาดจากแผ่นดินใหญ่ไปสู่ตัวเกาะ

บนเกาะฟุคุอุระจิมะมีจุดชมวิวอยู่มากมาย แต่ละจุดจะมีป้ายบรรยายไว้ด้วยว่าเกาะต่างๆที่เราเห็นนั้นมีความเป็นมาอย่างไรบ้าง

Location :

  • 39-1 มัตสึชิมะ, เขตมิยางิ, เมืองมัตสึชิมะ, จังหวัดมิยางิ, ประเทศญี่ปุ่น 981-0213

How to Travel :

  • รถไฟและรถประจำทาง : จากสถานี JR Sendai ให้ขึ้นรถไฟไปลงที่สถานี Matsushima Kaigan แล้วเดินต่อจากสถานีไปอีกประมาณ 15 นาที และนั่งเรือต่อไปที่เกาะฟุคุอุระจิมะ

Contact :

  • 022-354-3457

Hours :

  • เปิดทำการทุกวัน : 9:00 - 15:30 น.

Entrance Fee :

  • ผู้ใหญ่ : 200 เยน
  • เด็ก และ ผู้พิการ : 100 เยน
